# 💖 Para Você com Amor

Um projeto web interativo criado para transformar momentos especiais em uma experiência visual única.

## ✨ Sobre o Projeto

Este projeto é uma página romântica com galeria de fotos animada, mensagens interativas e efeitos visuais modernos, desenvolvida com foco em **experiência do usuário (UX)** e **design emocional**.

A ideia é simples: criar algo bonito, leve e marcante usando apenas HTML, CSS e JavaScript.

---

## 🚀 Funcionalidades

* 📸 Galeria de fotos com transições suaves
* ⏯️ Controle de navegação (anterior, próximo, play/pause)
* 💌 Mensagens animadas com efeito de digitação
* 🎵 Música de fundo com controle interativo
* ✨ Partículas animadas no fundo
* 💡 Efeito de luz seguindo o cursor (hover dinâmico)
* 🎨 Design moderno com glassmorphism e animações

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (Animações, Gradientes, Glassmorphism)
* JavaScript (DOM, Eventos, Classes)
* Google Fonts
